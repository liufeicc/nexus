# Hermes Agent PyInstaller 打包指南

## 打包结果

| 二进制文件 | 大小 | 入口点 | 用途 |
|-----------|------|--------|------|
| `dist/hermes` | ~192MB | `hermes_cli/main.py` | CLI 控制台（setup/chat/status/gateway 等所有子命令） |
| `dist/hermes-agent` | ~192MB | `run_agent.py` | 直接运行 Agent（接收 `--query` 参数） |

## 使用方式

```bash
./dist/hermes setup                    # 初始化配置（设置 API Key、模型等）
./dist/hermes chat                     # 进入交互式聊天
./dist/hermes --help                   # 查看所有子命令

./dist/hermes-agent --query "你好"     # 一次性运行 Agent
./dist/hermes-agent --help             # 查看 Agent 参数
```

## 打包环境

- Python 3.11（conda 环境 `hermes`）
- PyInstaller 6.19.0
- 已安装 Hermes 所有依赖：`pip install .[all]`

## 快速打包

## Spec 文件说明

### hermes.spec（CLI）

```python
from PyInstaller.utils.hooks import collect_all

datas = []
binaries = []
hiddenimports = [
    'hermes_cli', 'hermes_cli.main', 'hermes_cli.config', 'hermes_cli.setup',
    # ... 所有 hermes_cli 子模块 ...
    'tools', 'tools.registry',
    # ... 所有 tools 子模块 ...
    'gateway', 'cron', 'plugins', 'agent', 'fire',
]
tmp_ret = collect_all('hermes_cli')
datas += tmp_ret[0]; binaries += tmp_ret[1]; hiddenimports += tmp_ret[2]
tmp_ret = collect_all('firecrawl')
datas += tmp_ret[0]; binaries += tmp_ret[1]; hiddenimports += tmp_ret[2]

a = Analysis(
    ['hermes_cli/main.py'],

        'torch', 'tensorflow', 'matplotlib',

        'skimage', 'astropy', 'scipy', 'sklearn',

    ],
    # ...
)
```

## 体积优化说明

通过排除不必要的依赖，打包体积从 **699MB 降至 192MB**：

| 排除的库 | 原因 |
|---------|------|
| torch | ML 训练，仅用于 RL 训练工具 |
| tensorflow | 同上 |
| matplotlib / scipy / sklearn / skimage | 数据分析，非运行时必需 |
| cv2 / PIL | 图像处理，可选功能 |
| pandas / jieba / wordcloud | 数据处理，非运行时必需 |
| PyQt5/6 / PySide2/6 / tkinter | GUI 库，Hermes 是 CLI |
| pytest / debugpy | 开发测试，非运行时必需 |
| imageio / astropy | 科学计算，非运行时必需 |

## 跨平台打包

PyInstaller **不支持跨平台编译**，必须在目标平台上分别打包：

| 平台 | 打包环境 | 输出文件 |
|------|---------|---------|
| Linux | Ubuntu 24.04 x86_64 | `dist/hermes-agent`（ELF） |
| Windows | Windows 10/11 x64 | `dist/hermes-agent.exe`（PE） |
| macOS | macOS x64/ARM64 | `dist/hermes-agent`（Mach-O） |

打包流程与 Linux 完全相同，只需在对应平台上执行即可。
